## 🚀 快速部署

### 1️⃣ 环境要求
- **PHP**: 7.4+
- **MySQL**: 5.7+ 或 MariaDB 10.2+
- **扩展**: pdo_mysql

### 2️⃣ 一键安装
1. **上传文件**到网站根目录
2. **访问安装向导**: `http://yourdomain.com/install.php`
3. **填写数据库信息**完成安装
4. **访问后台**: `http://yourdomain.com/admin/`

### 3️⃣ 默认账号
- **用户名**: admin
- **密码**: admin123

## 🔌 API接口

### 获取网站数据
```
GET /api.php
```

### 按分类获取链接
```
GET /api.php?category=技术
```

### 搜索链接
```
POST /api.php
Content-Type: application/json

{"action": "search", "keyword": "设计"}
```
